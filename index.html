Stap C: Firebase (De Database)

De connectie: GitHub bewaart de code, Firebase bewaart de data (antwoorden van studenten). 🔥 Open Firebase Console

Actie: Realtime Database activeren

Klik in Firebase op Build > Realtime Database en kies 'Create Database'.

Stap C: Koppelen & Regels

⚠️ Stap 1: Beveiligingsregels aanpassen
Ga naar het tabblad Rules in Firebase, kopieer onderstaande tekst en plak deze over de bestaande regels heen. Klik daarna op Publish.

  "rules": {
    ".read": true,
    ".write": true
  }
