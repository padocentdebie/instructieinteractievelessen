Stap E: Publiceren op internet

Instructie:

kopieer de code die Gemini maakte.

  • Open tabblad met Github
  • klik op 'new' bij 'Repositories'
  • Bedenk een handige naam voor jouw werkvormwebsite en klik op 'create repository'
  • klik op 'creating a new file' , type in het invoer veld 'index.html' (Altijd 'index.html' noemen!)
  • plak de code in het veld en klik op 'commit'
  • klik op 'settings' (menubalk boven)
  • klik op 'pages' (menubalk links, een van de onderste knoppen)
  • klik onder 'branch' 'main'en dan 'save'

minuutje wachten en jouw les is beschikbaar op internet. Er verschijnt een blauwe link in het veld.
