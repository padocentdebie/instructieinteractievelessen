Stap 1: GitHub & Veiligheid

GitHub is de online plek waar we de code van onze les opslaan. Voordat we beginnen, een heel belangrijke afspraak:

⚠️ Belangrijke Veiligheidsinstructie

Plaats NOOIT politiegegevens of privacygevoelige informatie in de AI. Houd je lessen altijd algemeen en educatief.

De eerste stap: Log in op GitHub.com via je Google-account en zorg voor een neutrale gebruikersnaam (geen politie-verwijzingen).

Stap 2: Een nieuwe lesmap aanmaken

In GitHub heet een projectmap een 'Repository'. Zo maak je deze aan:

1. Klik op de groene knop 'New' bij je Repositories.

2. Geef je les een duidelijke, neutrale naam en klik onderaan op 'Create repository'.

Stap 3: Het bestand maken

Nu de map klaar is, gaan we de code die Gemini voor ons heeft geschreven erin plakken.

1. Klik op de link 'creating a new file' in het blauwe vak.

2. Typ bij de bestandsnaam exact: index.html (rode pijl). Plak daarna de code van Gemini in het grote witte vlak (gele pijl).

Stap 4: Firebase (De Verbinding)

Firebase is de 'supercomputer' die zorgt dat de les interactief is. Het verbindt de laptops van de agenten in de klas met jouw scherm.

Log in op Firebase.google.com met Google en maak een nieuw project aan met dezelfde neutrale naam als op GitHub.

Stap 5: Plaatsen in It's Learning

Kopieer de iFrame-code (het venster). Ga naar It's Learning, kies 'Broncode' (het icoontje < >) en plak de code daar in.

Je interactieve les is nu klaar voor gebruik!
