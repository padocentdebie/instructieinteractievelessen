Stap B: Wat is GitHub?

Je hebt nu de code, maar die staat nog op je klembord. Om de les toegankelijk te maken voor je studenten, moet de code op internet geplaatst worden. Hiervoor gebruiken we GitHub.

Wat doet GitHub?
Zie GitHub als een beveiligde online archiefkast. Het host jouw code en maakt er een werkende website van met een unieke link.

De Opdracht: Log in op GitHub. Gebruik hiervoor je Google-account, dat is wel zo makkelijk.

🌐 Ga naar GitHub & Log in met Google

Tip: Kies voor "Sign in with Google" om direct toegang te krijgen zonder nieuw wachtwoord.

Stap C: Firebase (De Motor)

Zonder Firebase is je les een statische tekst. Firebase is de 'motor' die ervoor zorgt dat studenten live resultaten kunnen zien en antwoorden naar de database kunnen sturen.

Stap C: Firebase Instellen

Cruciale stap: Zet in de Firebase Console onder het tabblad 'Rules' de waarden van 'read' en 'write' op true. Klik daarna op 'Publish'.
