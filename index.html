Stap 1: GitHub (Je Online Archiefkast)

GitHub is de plek waar jouw interactieve les "woont" op het internet. Omdat Gemini een technische code voor je schrijft, hebben we een plek nodig die deze code begrijpt en toegankelijk maakt voor de laptops van je studenten. GitHub bewaart je werk en geeft je een link die je later in It's Learning plakt.

⚠️ Belangrijke Veiligheidsinstructie

Plaats NOOIT politiegegevens, vertrouwelijke informatie of privacygevoelige gegevens van burgers in de AI (Gemini). Gebruik uitsluitend algemene, educatieve termen voor je lessen om de dataveiligheid te waarborgen.

Inloggen en je account instellen:

  • Ga naar GitHub.com en kies voor 'Sign up'.
  • Gebruik de knop 'Continue with Google' om in te loggen met je werk-account.
  • Kies een neutrale gebruikersnaam zonder verwijzingen naar de politie (bijv. Onderwijs_Project_2026).
  • Bevestig de autorisatie en controleer je Gmail voor de verificatiecode.

Stap 2: Firebase (De Supercomputer)

Waar GitHub de opslag is, is Firebase het brein dat alles live verbindt. Zonder Firebase zou een Bingo-spel niet werken; het zorgt er namelijk voor dat als jij op "Volgend Woord" klikt, alle schermen in de klas tegelijkertijd verspringen. Het is de motor achter de interactiviteit.

Hoe activeer je de verbinding?

  • Bezoek Firebase.google.com en klik op 'Get started'.
  • Log in via de 'Google' knop met hetzelfde account als bij GitHub.
  • Maak een nieuw project aan met een neutrale naam.
  • Na het aanmaken kom je in de 'Console'. Dit is het hart van je live-verbinding.

Stap 3: Gemini & Het "Prompten"

Gemini is je technische assistent. Jij hoeft geen programmeur te zijn; Gemini vertaalt jouw onderwijskundige wensen naar een digitaal spel. Het proces van een vraag stellen aan de AI noemen we "prompten".

Om een resultaat te krijgen waar je echt iets aan hebt, moet je uitgebreid zijn in je uitleg:

  • Context: Vertel voor welk vak en welk niveau de les is.
  • Doel: Leg uit wat de studenten moeten leren of doen.
  • Functionaliteit: Beschrijf precies wat er moet gebeuren (bijv. "De student moet op een knop kunnen klikken en dan moet het vakje van kleur veranderen").

Stap 4: Je les tonen in It's Learning

De laatste stap is het plaatsen van je creatie in It's Learning. Dit doen we via een iFrame. Dit is eigenlijk een venster op je It's Learning pagina waar de inhoud van je GitHub-pagina doorheen schijnt.

De iFrame plaatsen:

  • Kopieer de iFrame-code die Gemini voor je heeft gegenereerd (begint met <iframe...).
  • Ga in It's Learning naar je vak en voeg een 'Pagina' toe.
  • Zoek in de tekstverwerker naar het icoontje met de haakjes < > (Broncode).
  • Plak de code onderaan en klik nogmaals op < > om het resultaat te zien.
  • Sla de pagina op en je interactieve les is live voor je studenten!
