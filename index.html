Stap E: Publiceren op internet

Kopieer de volledige code die Gemini zojuist voor je heeft gegenereerd. We gaan deze nu "hosten" zodat je studenten erbij kunnen.

  • Open het tabblad met GitHub.
  • Klik aan de linkerkant op de groene knop 'New' bij 'Repositories'.
  • Bedenk een handige naam voor jouw werkvorm (bijv. 'woordenschat-bingo') en klik onderaan op 'Create repository'.
  • Klik in het midden van het scherm op de blauwe link 'creating a new file'.
  • Typ in het invoerveld voor de bestandsnaam: index.html
    (Let op: Het MOET precies 'index.html' zijn, anders werkt de website niet!)
  • Plak de volledige code van Gemini in het grote tekstveld en klik rechtsboven op de groene knop 'Commit changes...' en daarna nogmaals op 'Commit changes'.
  • Ga nu in de bovenste menubalk naar 'Settings' (het tandwiel-icoon).
  • Klik in de linkerkolom op 'Pages' (onder het kopje Code and automation).
  • Kies onder 'Branch' voor 'main' en klik op de knop 'Save'.
Bijna klaar! Het duurt ongeveer een minuutje voordat GitHub je website heeft gebouwd. Er verschijnt straks bovenaan een balk met een link: "Your site is live at...".
